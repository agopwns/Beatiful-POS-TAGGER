# Beatiful POS-TAGGER
오타나 띄어쓰기, 문법 오류에 강한 품사 태깅 SW입니다.  
  
자연어 처리를 공부하며 진행하였으며 간단한 학습과 사전 추가, 자연어 처리 라이브러리를 융합하여 연구 목적으로 개발하였습니다.  
     
작품의 상세 작동 영상은 [여기](https://youtu.be/ybyop5Vn12c)에서 보실 수 있습니다.

## 기능
  - 오탈자 전처리
  - 띄어쓰기 전처리
  - 문법 오류 전처리
  - 품사 태깅

## 사용 기술
  - AWS EC2 
  - Flask 
  - Mecab 
  - soyspacing 
  - Hanspell 
  - Beatiful Soup 
  - HTML/CSS/JS/JQuery
